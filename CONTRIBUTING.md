# Contributing to react-step-form

Thanks for your interest in contributing.

## Development setup

1. Install dependencies: `npm install`
2. Run all checks: `npm run check`
3. Build all packages: `npm run build`
4. Start example app: `npm run dev:example`

## Pull request expectations

- Keep pull requests focused and small.
- Include tests for behavior changes.
- Update README/API docs when user-facing behavior changes.
- Use clear commit messages and PR descriptions.

## Code quality

- TypeScript strict mode is enabled.
- Public API changes should include type tests or runtime tests.
- Avoid introducing breaking changes without discussion in an issue.

## Reporting bugs

Open an issue and include:

- expected behavior
- actual behavior
- minimal reproduction
- environment details
